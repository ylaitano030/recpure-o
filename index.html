Comando CSS

O CSS é responsável pela formatação visual da página. O exemplo acima mostra como estilizar elementos HTML usando seletores. h1, h2 e p são seletores que aplicam estilos específicos aos títulos e parágrafos. font-family define a fonte, font-size define o tamanho do texto e color define a cor do texto. Aqui está um exemplo básico de um arquivo CSS:

        body {
            font-family: Arial, sans-serif;
        }
        h1 {
            font-size: 30px;
            color: blue;
        }
        h2 {
            font-size: 20px;
            color: red;
        }
        p {
            font-size: 15px;
            font-family: "Times New Roman", serif;
            color: black;
        }
