Os Principais Comandos para Criar um Site

Comando HTML

O HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele define a estrutura do conteúdo. Aqui está um exemplo básico:

        <!DOCTYPE html>
        <html lang="pt-BR">
        <head>
            <meta charset="UTF-8">
            <title>Título da Página</title>
        </head>
        <body>
            <h1>Título Principal</h1>
            <p>Este é um parágrafo de exemplo.</p>
        </body>
        </html>

Comando CSS

O CSS (Cascading Style Sheets) é usado para estilizar a aparência das páginas da web criadas com HTML. Aqui está um exemplo básico de um arquivo CSS:

        body {
            font-family: Arial, sans-serif;
        }
        h1 {
            font-size: 30px;
            color: blue;
        }
        h2 {
            font-size: 20px;
            color: red;
        }
        p {
            font-size: 15px;
            font-family: "Times New Roman", serif;
            color: black;
        }
